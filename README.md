# React + Vite + DaisyUI + Laravel APi

Installation Guide
1. git clone [Repository Link ]
2. cd my-project
3. npm i
4. npm install -D tailwindcss@3 postcss autoprefixer
5. npx tailwindcss init -p

এখন DaisyUI ইনস্টল করুন:
6. npm install daisyui

7. npm run dev
